I was able to create this icon using code.org's encoding color images widget. In this widget, I adjusted the image width to 19 pixels and the image height to 19 pixels to get a 19 x 18 image; I also adjusted the bits per image to 12 to get a pixel data made up of 3 hexadecimals. I then used a website with triple hex color codes to help color my image of a stethoscope, which signifies the medical field, my future career. In this activity I was able to learn how to create and encode color images by creating image dimensions and using a unique set of color pixel data. The significance of the icon is that it represents something that relates to me, since it is a significant symbol of doctors, but in general, the medical field, a career that is improtant to my family and my future.
